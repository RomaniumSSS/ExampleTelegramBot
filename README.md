# Vibe Tracker Telegram Bot

Простой бот для ежедневного трекинга уровня энергии и настроения. Позволяет сохранять оценки состояния и заметки, а также просматривать статистику за последние дни.

## Предварительные требования

Для работы бота на вашем компьютере должен быть установлен Python версии 3.11 или выше.

1. Проверьте, установлен ли Python. Откройте терминал (командную строку) и введите:
   ```bash
   python --version
   ```
   Если вы видите ошибку или версию ниже 3.11, скачайте и установите Python с официального сайта python.org.
   *Важно: При установке на Windows обязательно поставьте галочку "Add Python to PATH".*

## Установка и настройка

Выполняйте шаги последовательно. Все команды вводятся в терминале (командной строке) в папке с проектом.

### 1. Создание виртуального окружения

Виртуальное окружение необходимо для того, чтобы библиотеки проекта не конфликтовали с другими программами на вашем компьютере. Оно создает изолированную папку `venv`.

**Для Windows:**
```cmd
python -m venv venv
```

**Для macOS и Linux:**
```bash
python3 -m venv venv
```

### 2. Активация виртуального окружения

После создания окружения его нужно активировать. После активации в начале строки терминала должна появиться надпись `(venv)`.

**Для Windows:**
```cmd
venv\Scripts\activate
```

**Для macOS и Linux:**
```bash
source venv/bin/activate
```

### 3. Установка зависимостей

Установите необходимые библиотеки, указанные в файле `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 4. Регистрация бота и получение токена

Чтобы бот заработал, его нужно создать в Telegram и получить уникальный ключ доступа (токен).

1. Откройте Telegram и в поиске найдите бота **@BotFather** (у него должна быть синяя галочка).
2. Нажмите кнопку "Запустить" или отправьте команду `/start`.
3. Отправьте команду `/newbot` для создания нового бота.
4. Введите имя бота (как он будет отображаться в списке чатов, например: `Мой Вайб Трекер`).
5. Введите техническое имя пользователя (username). Оно должно быть уникальным, писаться на латинице и обязательно заканчиваться на `bot` (например: `MyVibeTracker_bot`).
6. Если имя свободно, BotFather пришлет сообщение с токеном. Это длинная строка вида `123456789:ABC-DefGhIjKlMnOpQrStUvWxYz`. Скопируйте этот токен.

### 5. Настройка конфигурации (.env)

Бот не хранит секретные данные (токены) в открытом коде. Их нужно поместить в специальный файл настроек.

1. В папке проекта создайте новый файл и назовите его `.env` (точка в начале обязательна).
2. Откройте этот файл любым текстовым редактором (Блокнот, Notepad++, VS Code).
3. Запишите в него следующую строку, подставив свой токен, полученный от BotFather:

```env
BOT_TOKEN=ваш_скопированный_токен
```

Пример содержимого файла `.env`:
```env
BOT_TOKEN=705678901:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
```

## Запуск бота

Убедитесь, что виртуальное окружение активировано (см. шаг 2) и все настройки выполнены.

Запустите бота командой:

```bash
python main.py
```

Если все сделано верно, в консоли появится сообщение о запуске. Теперь вы можете найти своего бота в Telegram по юзернейму и нажать `/start`.

## Остановка бота

Чтобы остановить работу бота, в терминале нажмите сочетание клавиш `Ctrl + C`.
